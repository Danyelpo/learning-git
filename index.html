Learning Git

What is Git?

Git is a version control system that tracks changes in files. It allows returning to previous versions, working safely, and collaborating with others.

Why Git is Important?

  • Tracks code history
  • Prevents losing work
  • Enables teamwork
  • Industry standard for developers

Git vs GitHub

Git GitHub
Tools Platform
Runs locally Runs online
Tracks changes Hosts repositories

Git basic commands

1. git init - Initialized empty Git repository in "user file path/.git/".

What happens when git init is executed?

  • A hidden .git folder is created
  • This folder contains:
    • Commit history
    • Branch information
    • Configuration files
  • The project becomes a Git repository
