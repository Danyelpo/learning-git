# Learning Git

A beginner-friendly project for learning **Git and GitHub** through practice and documentation.  
This repository also serves as a simple website explaining Git concepts.

## Purpose
- Learn Git by actually using it
- Practice real Git workflows (branches, commits, merges)
- Document Git concepts in a simple and clear way
- Build a small documentation-style website
